# basic HTML Website

A Basic Multiple-page Website built with semantic HTML.

## Live Project URL
http://127.0.0.1:5500/pages/homepage.html


## Features
- Semantic HTML Structure
- multiple pages - homepage, project, articles, contact
- Seo meta tags for better search visibility
- Open Graph tags for Social sharing
- Favicon included


## How to Use
Open `index.html` in any browser to view the website.

## Project Structure
homepage.html  
projects.html
articles.html
contact.html
img/jd (1).png (or your favicon file)

## Purpose
Practice building clean, accessible, well-structured Basic HTML Website.
